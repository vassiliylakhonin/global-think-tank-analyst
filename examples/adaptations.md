# Adaptation Blueprints

Use these as fork starting points.

## 1) Government Policy Desk

- Rename skill to: `policy-risk-analyst`
- Emphasize: diplomatic options, sanctions exposure, escalation pathways
- Add outputs: policy memo, implementation sequencing, interagency risk notes

## 2) Startup / VC Strategy

- Rename skill to: `market-strategy-analyst`
- Emphasize: TAM shifts, competitor response, regulatory constraints, GTM timing
- Add outputs: strategic bets, downside scenarios, trigger-based pivots

## 3) Nonprofit Program Strategy

- Rename skill to: `impact-policy-analyst`
- Emphasize: donor alignment, delivery risk, safeguarding, local political economy
- Add outputs: concept-note framing, risk register, indicators for adaptation
